# Hidden Gem Music App

React Native app structured for web first, while keeping the same component tree usable for mobile.

## Next Inputs

1. Put your Mapbox token into `.env` as `EXPO_PUBLIC_MAPBOX_ACCESS_TOKEN=...`
2. Put your font files into `src/assets/fonts/`
3. If you want exact font hookup next, rename them to:
   - `EditorialSerif-Regular.ttf`
   - `EditorialSerif-Italic.ttf`
   - `CondensedSans-Regular.ttf`
   - `CondensedSans-Bold.ttf`

## Install

```bash
npm install
npm run web
```
